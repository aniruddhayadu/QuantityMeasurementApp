**UC7: Addition with Target Unit Specification**

 - Adds two lengths with explicit target unit.
 - Method: add(Length other, LengthUnit targetUnit).
 - Provides flexibility to return result in any supported unit.
 - Preserves immutability, commutativity, and precision across unit scales.
 - Demonstrates method overloading for backward compatibility.
